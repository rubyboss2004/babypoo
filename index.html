歡迎來到寶寶勇士樂譜庫

這裡有老師整理的詩歌譜，給寶寶勇士的樂手練習用。第一次來嗎？花一分鐘看看怎麼用～

怎麼看譜？

在主頁上看到歌曲清單，點一下歌名就會展開譜面。再點一次就會收起來。

怎麼下載到手機/電腦？

展開譜面後，下面有「下載 / 存到相簿」按鈕，按下去就會跳出選單。

  • 手機：選「儲存圖片」存到相簿
  • 電腦：圖片會直接下載到電腦

找不到想要的歌怎麼辦？

按上方的「我想要新的譜」，輸入歌名送出，老師會收到通知並盡快寫好上傳。

把網頁加到主畫面，下次更快打開

加到主畫面後，就像 App 一樣點一下就能用，不用再開瀏覽器找書籤～

iPhone（Safari）
  1. 用 Safari 打開這個網頁（不要用 LINE 內建瀏覽器）
  2. 按螢幕下方中間的「分享」按鈕（向上箭頭的圖示）
  3. 往下捲，選「加入主畫面」
  4. 確認名稱後按右上角的「新增」
Android（Chrome）
  1. 用 Chrome 打開這個網頁
  2. 按右上角的「⋮」（三個點）
  3. 選「加到主畫面」或「安裝應用程式」
  4. 按「新增」就完成了
電腦（Chrome / Edge）
  1. 網址列右邊找到「安裝」圖示（一個方框加箭頭）
  2. 按下去 → 點「安裝」
  3. 之後桌面就會有捷徑可以直接打開

如果沒看到安裝按鈕，加到書籤也可以喔～
